# Guthi Network 
Network part of the **Guthi-Distribution** project done as a final year project of undergraduate level. 

### Requirements
* Language: Go 1.19, [TDM-GCC](https://jmeubank.github.io/tdm-gcc/)
* Build System: CMake, Ninja or Makefile


### Build Instruction 
```
go build
./GuthiNetwork
```
